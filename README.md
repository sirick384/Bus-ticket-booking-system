# Bus-ticket-booking-system

RideBus is a simple Java Swing-based desktop application developed for booking bus tickets easily. In this project, users can log in, search buses by selecting source, destination, and date, check available seats, view ticket prices, and book tickets by entering passenger details. After booking, the system generates a confirmation with a PNR number. This project is built using Java, Java Swing, AWT, and Object-Oriented Programming concepts to provide a user-friendly interface and simulate a real-world bus booking system like RedBus and KSRTC.
